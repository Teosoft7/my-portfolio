# OOHLIB: Spatial Data Analysis Platform

Out-Of-Home (OOH) advertising evaluation is historically difficult. This project introduces a data-driven approach to estimate the impact of billboard advertisements based on spatial intelligence and ML.

## The Problem
Measuring the actual conversion rate or impressions of physical billboards is highly inaccurate compared to digital ads.

## The Solution
We ingested massive amounts of spatial data, including foot traffic, vehicular movement, and demographic heatmaps, to create a custom ML scoring algorithm.

### Architecture
- **Backend:** FastAPI for high-performance API endpoints.
- **Frontend:** Next.js for interactive mapping and dashboards.
- **Database:** PostgreSQL with PostGIS for spatial queries.
